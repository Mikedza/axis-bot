# Axis Bot - Multi-Personality AI Companion Bot
Axis is a multi-personality AI Discord bot that adapts to users through different behavioral “axes.”
It remembers conversations, switches personalities seamlessly, and provides dynamic emotional and conversational support.

Full Description:
Axis is a Discord-based AI companion built around the idea of a central system that branches into multiple directions — like an axis or guiding star. Each direction represents a unique personality, allowing users to choose how they want to interact, whether for motivation, support, or humor.

The bot maintains persistent, per-user memory, enabling continuous and natural conversations instead of isolated responses. Users can switch between personalities at any time while preserving context, creating a flexible and personalized experience.

Axis supports both public and private interactions and is powered by a local AI model, allowing unlimited usage without external API costs. Overall, it acts as a multi-dimensional assistant that evolves with the user and adapts to their needs.
